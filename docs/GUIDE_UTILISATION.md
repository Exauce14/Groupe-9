# Guide d'utilisation non technique

## 1. Objectif du guide

Ce document explique clairement comment utiliser l'application bancaire sans entrer dans des détails techniques. Il est destiné à un utilisateur final qui souhaite comprendre le fonctionnement général et les principales actions possibles.

## 2. Accéder à l'application

1. Ouvrez votre navigateur.
2. Rendez-vous sur l'adresse de l'application (par exemple `http://localhost:3000` ou l'URL fournie par le projet).
3. La page d'accueil présente les options principales : connexion, inscription et informations sur les services.

## 3. Inscription

Pour utiliser l'application, créez d'abord un compte :

1. Cliquez sur le bouton `S'inscrire`.
2. Remplissez le formulaire avec :
   - votre adresse email
   - un mot de passe sécurisé
   - votre prénom et nom
   - votre numéro de téléphone
   - votre adresse
   - votre date de naissance
   - votre genre
   - votre revenu annuel
   - votre type de résidence
   - votre numéro d'assurance sociale (SIN)
3. Validez le formulaire.
4. Vous recevez un code par email pour la vérification 2FA.
5. Entrez ce code sur la page prévue pour finaliser l'inscription.

Après validation, votre compte est créé et votre dossier est enregistré.

## 4. Connexion

1. Cliquez sur `Connexion`.
2. Saisissez votre adresse email et votre mot de passe.
3. Si les informations sont correctes, un code 2FA est envoyé par email.
4. Entrez le code reçu pour accéder à votre espace personnel.

Une fois connecté, vous pouvez accéder à votre tableau de bord et à toutes les fonctionnalités autorisées.

## 5. Tableau de bord et navigation

Après connexion, vous arrivez sur le tableau de bord. Depuis ce point, vous pouvez :

- consulter vos comptes bancaires
- voir votre solde
- accéder à votre historique de transactions
- gérer vos cartes
- envoyer et recevoir des transferts
- payer des factures
- consulter les notifications
- faire des demandes de services

## 6. Gestion des comptes

### Voir vos comptes

- Ouvrez la section `Mes comptes`.
- Vous voyez la liste de vos comptes avec leur numéro, type et solde.
- Cliquez sur un compte pour obtenir plus de détails.

### Créer un nouveau compte

- Allez dans `Créer un compte`.
- Choisissez le type de compte (chèque, épargne, crédit, investissement).
- Validez la création.

## 7. Gestion des cartes

### Voir vos cartes

- Ouvrez la section `Mes cartes`.
- Vous voyez toutes les cartes associées à vos comptes.

### Bloquer une carte

- Sélectionnez la carte souhaitée.
- Cliquez sur `Bloquer` si elle est perdue ou compromise.

## 8. Transactions et virements

### Voir l'historique

- Ouvrez la section `Historique des transactions`.
- Vous pouvez voir toutes les opérations sur vos comptes.

### Faire un virement interne

- Ouvrez `Virement entre comptes`.
- Choisissez le compte source et le compte destinataire.
- Indiquez le montant.
- Confirmez la transaction.

## 9. Transferts Interac

### Envoyer un transfert

- Allez dans `Transferts Interac`.
- Choisissez un bénéficiaire ou ajoutez-en un nouveau.
- Indiquez le montant et la question de sécurité.
- Envoyez le transfert.

### Recevoir un transfert

- Vous recevez un message indiquant qu'un transfert vous est destiné.
- Ouvrez la section `Transferts reçus`.
- Entrez la réponse à la question de sécurité pour accepter le transfert.

## 10. Paiement de factures

### Payer une facture

- Allez dans `Paiement de factures`.
- Choisissez un fournisseur (Hydro-Québec, Bell, SAAQ, etc.).
- Indiquez le numéro de référence et le montant.
- Confirmez le paiement.

### Voir vos paiements

- Ouvrez `Mes paiements`.
- Consultez l'état de chaque facture : payée, en attente, rejetée.

## 11. Paiements récurrents

- Ouvrez `Paiements planifiés`.
- Programmez un paiement une seule fois ou en mode récurrent : hebdomadaire, bihebdomadaire ou mensuel.
- Choisissez la date de début et le compte source.

## 12. Notifications

- Ouvrez `Notifications`.
- Vous verrez les alertes importantes : virement effectué, facture due, carte bloquée.
- Marquez les notifications comme lues pour garder votre espace clair.

## 13. Demandes de service

### Faire une demande

- Ouvrez `Demandes`.
- Sélectionnez le type de demande (nouveau compte, carte, prêt, service).
- Ajoutez une description et envoyez.

### Suivre une demande

- Consultez `Mes demandes`.
- Vérifiez l'état : en attente, approuvée, rejetée, annulée.

## 14. Déconnexion

- Cliquez sur le bouton `Déconnexion`.
- Votre session se termine.
- Pour revenir, reconnectez-vous avec votre email et mot de passe.

## 15. Support et erreurs courantes

### Que faire si une action échoue

- Vérifiez vos informations saisies.
- Assurez-vous que le code 2FA est correct et encore valide.
- Si le message indique un problème d'autorisation, reconnectez-vous.
- En cas d'erreur serveur, réessayez plus tard.

### Recommandations

- Conservez vos codes et mots de passe en lieu sûr.
- Ne partagez jamais votre code 2FA.
- Vérifiez régulièrement vos notifications pour ne rien manquer.

## 16. Résumé

L'application permet de gérer vos comptes bancaires, cartes, virements, transferts Interac et paiements de factures de manière simple. La navigation principale se fait depuis le tableau de bord, et chaque fonctionnalité importante propose un écran clair pour saisir les informations et confirmer l'action.
