## 1. Presentation de l'application

La Plateforme Bancaire Digitale est une application de banque en ligne qui vous permet de gerer votre argent depuis votre ordinateur ou votre navigateur web. Elle remplace les visites en succursale pour la grande majorite des operations courantes.

Avec cette application, vous pouvez :

- Consulter le solde de vos comptes en temps reel
- Effectuer des virements entre vos comptes
- Envoyer et recevoir des transferts d'argent par courriel (Interac)
- Payer vos factures courantes (Hydro-Quebec, Bell, SAAQ, etc.)
- Bloquer ou debloquer vos cartes en cas de perte
- Demander une nouvelle carte ou l'ouverture d'un compte supplementaire
- Recevoir des alertes instantanees pour chaque operation sur votre compte

## 2. Creer un compte

Avant de pouvoir utiliser l'application, vous devez creer un compte. Ce processus est entierement en ligne.

### Etapes

1. Ouvrez l'application dans votre navigateur. La page d'accueil s'affiche.
2. Cliquez sur le bouton **"Ouvrir un compte"** ou **"S'inscrire"**.
3. Remplissez le formulaire d'inscription avec les informations suivantes :
   - Votre prenom et nom de famille
   - Votre adresse courriel (ce sera votre identifiant de connexion)
   - Un mot de passe (choisissez un mot de passe fort, avec des lettres, chiffres et caracteres speciaux)
   - Votre numero de telephone
   - Votre date de naissance
   - Votre adresse postale complete
   - Votre statut professionnel (etudiant, employe, professionnel, retraite)
   - Votre revenu annuel approximatif
   - Votre type de residence (proprietaire ou locataire)
   - Votre numero d'assurance sociale (NAS)

4. Cliquez sur **"Soumettre ma demande"**.

### Ce qui se passe ensuite

Votre demande est envoyee a notre equipe administrative. Un administrateur examinera votre dossier et activera votre compte, generalement dans un delai de un a deux jours ouvrables. Vous recevrez un courriel de confirmation des que votre compte sera approuve ou, le cas echeant, si des informations supplementaires sont requises.

**Important :** Vous ne pourrez pas vous connecter tant que votre compte n'aura pas ete approuve par un administrateur.

## 3. Se connecter

La connexion a la plateforme se fait en deux etapes pour assurer la securite de votre compte.

### Etape 1 - Saisie des identifiants

1. Sur la page d'accueil, cliquez sur **"Se connecter"**.
2. Entrez votre adresse courriel et votre mot de passe.
3. Cliquez sur **"Connexion"**.

### Etape 2 - Verification par courriel (authentification a deux facteurs)

1. Un code a six chiffres est envoye automatiquement a votre adresse courriel.
2. Ouvrez votre boite de reception et copiez ce code.
3. Saisissez le code dans le champ prevu et cliquez sur **"Verifier"**.
4. Vous etes maintenant connecte et redirige vers votre tableau de bord.

**Le code est valide pendant 15 minutes.** Si vous ne l'avez pas recu, verifiez votre dossier de courrier indesirable ou cliquez sur **"Renvoyer le code"**.

### En cas de probleme de connexion

- Si vous avez oublie votre mot de passe, utilisez le lien **"Mot de passe oublié"** sur la page de connexion pour demarrer la procedure de reinitialisation.
- Apres plusieurs tentatives de connexion echouees, votre compte peut etre temporairement verrouille pour des raisons de securite. Attendez quelques minutes avant de reessayer.

### Mot de passe oublie

1. Sur la page de connexion, cliquez sur **"Mot de passe oublié"**.
2. Entrez l'adresse courriel associee a votre compte.
3. Consultez votre boite de reception et copiez le code de reinitialisation recu par courriel.
4. Retournez dans l'application et saisissez le code ainsi que votre nouveau mot de passe.
5. Confirmez le nouveau mot de passe et cliquez sur **"Reinitialiser"**.

Une fois la reinitialisation terminee, vous pouvez vous reconnecter avec votre nouveau mot de passe.

### Changer mon mot de passe

1. Apres vous etre connecte, ouvrez le menu utilisateur ou la section de parametres de votre compte.
2. Cliquez sur **"Changer mon mot de passe"**.
3. Saisissez votre mot de passe actuel.
4. Entrez ensuite votre nouveau mot de passe.
5. Confirmez le nouveau mot de passe en le saisissant une deuxieme fois.
6. Validez l'operation.

Le mot de passe est mis a jour immediatement. Si les deux saisies ne concordent pas, l'application vous demandera de ressaisir le nouveau mot de passe.

## 4. Le tableau de bord

Le tableau de bord est la premiere page que vous voyez apres votre connexion. Il vous donne un apercu rapide de votre situation financiere.

### Ce que vous pouvez voir

- **Resume de vos soldes** : Le solde actuel de chacun de vos comptes (compte cheques, epargne, etc.) et le total consolide.
- **Acces rapide** : Des boutons directs vers les principales fonctions (virement, Interac, paiement de factures, etc.).
- **Notifications recentes** : Les dernieres alertes concernant votre compte.

### Navigation dans l'application

Le menu principal en haut ou sur le cote vous permet d'acceder a toutes les sections de l'application :

- Mes comptes
- Mes cartes
- Virements
- Interac
- Paiement de factures
- Depot / Retrait
- Mes demandes
- Notifications

## 5. Consulter ses comptes et transactions

### Voir ses comptes

1. Dans le menu, cliquez sur **"Mes comptes"**.
2. La liste de vos comptes s'affiche avec pour chacun :
   - Le type de compte (compte cheques, epargne, credit, investissement)
   - Le numero de compte (partiellement masque pour votre securite)
   - Le solde actuel
   - Le statut du compte (actif, inactif, gele)

### Voir l'historique des transactions

1. Cliquez sur un compte pour l'ouvrir.
2. L'historique des 100 dernieres transactions s'affiche, de la plus recente a la plus ancienne.
3. Pour chaque transaction, vous pouvez voir :
   - La date et l'heure
   - La description de l'operation
   - Le montant (en vert si c'est un credit, en rouge si c'est un debit)
   - Le solde apres la transaction
   - Le numero de reference de la transaction

## 6. Gerer ses cartes bancaires

### Voir ses cartes

1. Dans le menu, cliquez sur **"Mes cartes"**.
2. La liste de toutes vos cartes de debit et de credit s'affiche.
3. Pour chaque carte, vous pouvez voir le numero masque, la date d'expiration et le statut (active, bloquee).

### Bloquer une carte

Si vous avez perdu votre carte ou qu'elle a ete volee, bloquez-la immediatement.

1. Dans **"Mes cartes"**, trouvez la carte concernee.
2. Cliquez sur le bouton **"Bloquer cette carte"**.
3. Indiquez la raison (perte, vol, autre).
4. Confirmez l'action.

La carte est bloquee instantanement. Plus aucun paiement ne pourra etre effectue avec cette carte.

### Debloquer une carte

Si vous avez retrouve votre carte et souhaitez la reactiver :

1. Dans **"Mes cartes"**, trouvez la carte bloquee.
2. Cliquez sur **"Debloquer cette carte"**.
3. Confirmez l'action.

La carte est reactivee immediatement.

## 7. Effectuer un virement entre ses propres comptes

Cette fonction vous permet de deplacer de l'argent entre vos propres comptes (par exemple, de votre compte cheques vers votre compte epargne).

### Etapes

1. Dans le menu, cliquez sur **"Virements"**.
2. Selectionnez le **compte source** (celui a partir duquel vous prelevez l'argent).
3. Selectionnez le **compte de destination** (celui qui recevra l'argent).
4. Entrez le **montant** a transferer.
5. Ajoutez une description si vous le souhaitez (optionnel).
6. Cliquez sur **"Effectuer le virement"**.

Le transfert est immediat. Vous verrez le solde des deux comptes se mettre a jour instantanement et vous recevrez une notification de confirmation.

**Note :** Vous ne pouvez pas transferer un montant superieur au solde de votre compte source.

## 8. Envoyer et recevoir un transfert Interac

Le transfert Interac vous permet d'envoyer de l'argent a n'importe quelle personne disposant d'une adresse courriel, meme si elle n'est pas cliente de cette banque.

### Envoyer de l'argent

1. Dans le menu, cliquez sur **"Interac"** puis **"Envoyer"**.
2. Entrez l'adresse courriel du destinataire (ou selectionnez un contact dans votre liste de beneficiaires).
3. Entrez le montant a envoyer.
4. Ajoutez un message optionnel pour le destinataire.
5. Creez une **question de securite** et tapez la **reponse** (que vous communiquerez separement au destinataire par telephone, par exemple).
6. Selectionnez le compte a debiter.
7. Cliquez sur **"Envoyer"**.

Le destinataire recevra un courriel l'informant du transfert. L'argent sera retire de votre compte immediatement. Si le destinataire ne depose pas le transfert dans les 30 jours, le transfert expire et l'argent vous est automatiquement rembourse.

### Annuler un transfert envoye

Tant que le destinataire n'a pas depose le transfert, vous pouvez l'annuler.

1. Dans **"Interac"**, cliquez sur **"Mes envois"**.
2. Trouvez le transfert en question (statut : en attente).
3. Cliquez sur **"Annuler"**.

L'argent est remis sur votre compte instantanement.

### Recevoir de l'argent

Lorsque quelqu'un vous envoie un transfert Interac, vous recevez une notification dans l'application.

1. Dans **"Interac"**, cliquez sur **"Mes receptions"**.
2. Vous verrez les transferts en attente.
3. Cliquez sur **"Deposer"** a cote du transfert que vous souhaitez accepter.
4. Repondez correctement a la question de securite posee par l'expediteur.
5. Selectionnez le compte sur lequel vous souhaitez recevoir l'argent.
6. Confirmez le depot.

L'argent est ajoute a votre compte immediatement.

### Gerer ses beneficiaires

Pour ne pas avoir a saisir l'adresse courriel de vos contacts frequents a chaque fois :

1. Dans **"Interac"**, cliquez sur **"Mes beneficiaires"**.
2. Cliquez sur **"Ajouter un beneficiaire"**.
3. Entrez le nom, l'adresse courriel et une note personnelle (optionnel).
4. Cliquez sur **"Enregistrer"**.

## 9. Payer une facture

La plateforme vous permet de payer directement vos factures courantes en ligne.

### Fournisseurs disponibles

- Hydro-Quebec
- Videotron
- Bell Canada
- Virgin Plus
- SAAQ (immatriculation et permis de conduire)
- Intact Assurances
- TD Assurances
- Fortivia Assurances

### Etapes

1. Dans le menu, cliquez sur **"Paiement de factures"**.
2. Choisissez le fournisseur dans la liste.
3. Entrez votre **numero de client** ou **numero de compte** tel qu'il apparait sur votre facture.
4. Entrez le montant a payer.
5. Selectionnez le compte bancaire a debiter.
6. Ajoutez une description si souhaite.
7. Cliquez sur **"Payer"**.

Le paiement est soumis pour traitement. Il sera confirme dans les prochaines heures et vous recevrez une notification une fois le paiement traite.

---

## 10. Planifier un paiement recurrent

Si vous payez regulierement la meme facture, vous pouvez configurer un paiement automatique.

### Etapes

1. Dans **"Paiement de factures"**, cliquez sur **"Planifier un paiement"**.
2. Selectionnez le fournisseur.
3. Entrez votre numero de client et le montant.
4. Choisissez la frequence :
   - **Une seule fois** : pour un paiement futur unique
   - **Hebdomadaire** : tous les 7 jours
   - **Aux deux semaines** : toutes les deux semaines
   - **Mensuel** : une fois par mois, a la date de votre choix
5. Si vous choisissez mensuel, indiquez le **jour du mois** (entre le 1 et le 28).
6. Vous pouvez indiquer une date de fin (optionnelle). Si vous ne mettez pas de date de fin, le paiement continuera indefiniment jusqu'a ce que vous l'annuliez.
7. Cliquez sur **"Planifier"**.

### Annuler un paiement planifie

1. Dans **"Paiement de factures"**, cliquez sur **"Mes paiements planifies"**.
2. Trouvez le paiement a annuler.
3. Cliquez sur **"Annuler"**.

Le paiement ne sera plus execute a partir de la prochaine echeance.

## 11. Faire un depot ou un retrait en especes

Pour deposer ou retirer de l'argent en especes, vous devez soumettre une demande qui sera traitee par notre equipe.

### Demande de depot

1. Dans le menu, cliquez sur **"Depot / Retrait"**.
2. Cliquez sur **"Faire un depot"**.
3. Selectionnez le compte sur lequel vous souhaitez deposer l'argent.
4. Entrez le montant.
5. Ajoutez une description (optionnel).
6. Cliquez sur **"Soumettre la demande"**.

Un administrateur traitera votre demande. Apres validation, le montant sera credite sur votre compte et vous recevrez une notification de confirmation.

### Demande de retrait

La procedure est identique a celle du depot, mais vous cliquez sur **"Faire un retrait"**. Assurez-vous que votre compte dispose d'un solde suffisant avant de soumettre la demande.

## 12. Soumettre une demande de service

Vous pouvez demander l'ouverture de nouveaux comptes, l'emission de nouvelles cartes ou des produits de credit directement depuis l'application.

### Types de demandes disponibles

- **Ouverture de compte** : Demandez un compte epargne ou un compte de placement.
- **Carte de credit** : Demandez une nouvelle carte de credit en precisant la limite souhaitee.
- **Pret** : Demandez un financement en precisant le montant et la duree souhaitee.

### Etapes

1. Dans le menu, cliquez sur **"Mes demandes"**.
2. Cliquez sur **"Nouvelle demande"**.
3. Selectionnez le type de demande.
4. Remplissez le formulaire avec les informations requises.
5. Cliquez sur **"Soumettre"**.

### Suivre l'etat de ses demandes

1. Dans **"Mes demandes"**, vous pouvez voir toutes vos demandes en cours et passees.
2. Chaque demande a un statut :
   - **En attente** : Votre demande est en cours d'examen.
   - **Approuvee** : Votre demande a ete acceptee. Le service demande a ete cree.
   - **Rejetee** : Votre demande a ete refusee. Un commentaire explicatif est disponible.

Vous recevrez automatiquement une notification par l'application des que votre demande aura ete traitee.

## 13. Consulter ses notifications

Les notifications vous tiennent informe en temps reel de toute activite sur votre compte.

### Acces aux notifications

L'icone de cloche en haut de la page affiche le nombre de notifications non lues. Cliquez dessus pour les consulter.

Vous pouvez aussi aller dans **"Notifications"** dans le menu principal pour voir toutes vos notifications, y compris les plus anciennes.

### Types de notifications que vous pouvez recevoir

- Confirmation de virement effectue
- Reception d'un transfert Interac
- Approbation ou refus d'une demande de service
- Traitement d'un depot ou retrait
- Confirmation d'un paiement de facture
- Blocage ou deblocage d'une de vos cartes
- Suspension ou reactivation de votre compte par l'administration

### Marquer les notifications comme lues

- Cliquez sur une notification individuelle pour la marquer comme lue.
- Cliquez sur **"Tout marquer comme lu"** pour effacer tous les indicateurs de non-lu en une seule fois.

## 14. Espace administrateur

Cette section est reservee aux employes ayant un compte administrateur. Elle n'est pas accessible aux clients standard.

### Connexion en tant qu'administrateur

La procedure de connexion est identique a celle des clients (courriel, mot de passe, code 2FA). Apres connexion, vous etes automatiquement redirige vers le tableau de bord administrateur.

### Tableau de bord administrateur

Le tableau de bord affiche un resume des actions en attente :

- Nombre d'inscriptions a approuver
- Nombre de demandes de services en attente
- Nombre de transactions (depots/retraits) a traiter
- Nombre de paiements de factures en attente
- Statistiques generales de la plateforme (nombre d'utilisateurs actifs, solde total, etc.)

### Gestion des inscriptions

1. Cliquez sur **"Inscriptions en attente"**.
2. Examinez le dossier de chaque nouvel utilisateur.
3. Cliquez sur **"Approuver"** pour activer le compte. Un compte bancaire courant est automatiquement cree pour l'utilisateur, qui recut une notification par courriel et dans l'application.
4. Cliquez sur **"Rejeter"** si le dossier est incomplet ou non conforme. Vous pouvez laisser un commentaire qui sera transmis a l'utilisateur.

### Gestion des demandes de services

1. Cliquez sur **"Demandes en attente"**.
2. Examinez chaque demande (ouverture de compte, carte de credit, pret).
3. Approuvez ou rejetez en laissant un commentaire si necessaire.
4. En cas d'approbation, le service est cree automatiquement et l'utilisateur est notifie.

### Gestion des transactions

1. Cliquez sur **"Transactions en attente"**.
2. Examinez les demandes de depots et de retraits en attente.
3. Approuvez pour crediter ou debiter le compte de l'utilisateur.

### Gestion des paiements de factures

1. Cliquez sur **"Paiements en attente"**.
2. Examinez chaque paiement soumis par les utilisateurs.
3. Approuvez pour confirmer le paiement et notifier l'utilisateur.

### Gestion des utilisateurs

1. Cliquez sur **"Gestion des utilisateurs"**.
2. Recherchez un utilisateur par nom ou courriel.
3. Filtrez par statut (actif, en attente, suspendu).
4. Vous pouvez suspendre un compte utilisateur ou le reactiver depuis cet ecran.

## 15. Questions frequentes

**Je n'ai pas recu mon code de verification lors de la connexion. Que faire ?**

Verifiez d'abord votre dossier de courrier indesirable (spam). Si le code n'y est pas non plus, attendez une minute et cliquez sur **"Renvoyer le code"** sur la page de verification. Le code est valide pendant 15 minutes.

**Mon compte est en attente depuis plusieurs jours. Que se passe-t-il ?**

Le delai d'approbation est generalement de un a deux jours ouvrables. Si ce delai est depasse, contactez le support de votre institution pour obtenir une mise a jour sur l'etat de votre dossier.

**J'ai bloque ma carte par erreur. Puis-je la debloquer moi-meme ?**

Oui. Rendez-vous dans **"Mes cartes"**, trouvez la carte concernee et cliquez sur **"Debloquer"**. La carte est reactivee immediatement.

**Puis-je annuler un transfert Interac apres l'avoir envoye ?**

Oui, a condition que le destinataire n'ait pas encore depose le transfert. Allez dans **"Interac" > "Mes envois"** et cliquez sur **"Annuler"** a cote du transfert en question. Le montant sera remis sur votre compte immediatement.

**Mon transfert Interac a expire. Vais-je recuperer mon argent ?**

Oui. Lorsqu'un transfert Interac expire (apres 30 jours sans etre depose), le montant est automatiquement remboursé sur votre compte source. Vous recevrez une notification a ce sujet.

**Je veux changer mon mot de passe. Comment faire ?**

Vous pouvez utiliser la fonction **"Changer mon mot de passe"** depuis votre espace personnel apres vous etre connecte, ou demarrer la procedure de reinitialisation via le lien **"Mot de passe oublié"** sur la page de connexion si vous ne connaissez plus votre mot de passe actuel.

**L'application ne se charge pas correctement. Que faire ?**

Essayez de vider le cache de votre navigateur (Ctrl+Maj+Delete sur la plupart des navigateurs) et rechargez la page. Assurez-vous d'utiliser un navigateur recent (Chrome, Firefox, Edge ou Safari dans leur version la plus a jour).

**Mes informations personnelles ont change. Comment les mettre a jour ?**

Pour toute modification de vos informations personnelles (adresse, telephone, etc.), veuillez contacter directement votre conseiller bancaire ou le support de l'institution. Cette action necessite une validation de votre identite et ne peut pas etre effectuee en libre-service depuis l'application.
